# 專案根目錄 CLAUDE.md 實體化計畫

此計畫旨在將專案根目錄下的 `CLAUDE.md` 由軟連結改為實體檔案，內容直接同步自 `claude/CLAUDE.md`，以避免跨平台（如 Windows）使用時的相容性問題。

## 實作步驟

- [x] **步驟 1**：移除根目錄的 `CLAUDE.md` 軟連結
  - *說明*：刪除現有的 symlink 檔案。
- [x] **步驟 2**：建立實體檔案 `CLAUDE.md` 並寫入內容
  - *說明*：讀取 `claude/CLAUDE.md` 內容，寫入根目錄新建立的實體 `CLAUDE.md`。
- [x] **步驟 3**：更新 `@tree.md`
  - *說明*：更新根目錄 `CLAUDE.md` 的描述，標記其為實體檔案而非軟連結。
- [x] **步驟 4**：歸檔計畫檔案
  - *說明*：將此計畫移至 `.archive/` 資料夾中。
